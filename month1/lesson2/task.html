Below you will find the challenges. Carefully replicate the outcome, change only the markup—not the words. Remember to validate your code. Good luck!

Task One – Dual Headings

Create two level‑one headings that sit on the same line using inline‑block styling.
Hint: You can set display directly on the element.

Task Two – Rich Paragraph

Write a paragraph of three sentences. Inside it place an italic word, a red word, and a phrase with strong importance. End the paragraph with an inserted word and a deleted word followed by a highlight.

Task Three – Pre‑formatted Poem

Insert the following exactly as it appears:

Roses are red,
  Violets are blue,
    HTML is awesome,
      And so are you.

Task Four – Baseline Shifts

Display the formula H2O and the expression X2 inside a violet box.

Example: H2O   X2

Task Five – Quotations

“Code is like humor. When you have to explain it, it’s bad.”
“Programs must be written for people to read, and only incidentally for machines to execute.”

Recreate the two quotations above inside a violet container similar to the one in the previous task.

Task Six – Editable Heading

You should be able to click and edit the heading text in the browser.

Task Seven – Commentary

Add an HTML comment explaining the difference between block, inline, and inline‑block elements.
